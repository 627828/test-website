Adding an Image to Your Webpage

Using the <img> Tag

To add an image to your webpage, you can use the <img> tag with the src attribute specifying the URL of the image file.

This is a local image. Make sure the image file is stored in the same directory as your HTML file, and replace "example.jpg" with the name of your image file.

Image Attributes

The <img> tag supports several attributes:

  • src: Specifies the URL of the image file.
  • alt: Provides alternative text for the image, which is displayed if the image cannot be loaded or for accessibility purposes.
  • width: Specifies the width of the image in pixels or as a percentage.
  • height: Specifies the height of the image in pixels or as a percentage.
  • title: Adds a title to the image, which appears as a tooltip when the user hovers over the image.
